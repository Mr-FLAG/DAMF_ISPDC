# DAMF_ISPDC

Deploying compute resources at multiple geographically distributed sites has become common practice for exploiting data locality to meet strict latency and reliability requirements of big data services. We study utilizing network resources to enhance the fairness of compute resource allocation among jobs by transferring data and hence computation demands among sites. We define a new resource allocation policy called Dynamic Aggregate Max-min Fairness (DAMF), which requires the aggregate compute resource allocation to be max-min fair among all jobs under given network capacity constraints. We prove that DAMF is achievable and it satisfies widely accepted fairness properties including Pareto efficiency, envy-freeness and strategy-proofness. We present algorithms to compute the DAMF allocation and efficient heuristics to approximate it. Experimental results show that compared with existing allocation policies, DAMF can improve the balance of resource allocation and achieve higher system utility, especially when the job's workload distributions among sites are highly unbalanced.
